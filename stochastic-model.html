模型说明

Hasselmann (1967) 随机气候模型:

• 基本方程: dT/dt = -λT + η(t)

• λ: 负反馈系数，控制系统的阻尼和记忆时间

• η(t): 随机强迫，代表大气湍流等随机过程

• e-folding时间 = 1/λ，表示系统的特征时间尺度

• 该模型解释了海洋如何将大气的高频噪声转换为低频气候变化

特殊情况:

• λ = 0 (无负反馈): 系统变为随机游走，温度异常无限累积，无稳定状态

• σ = 0 (无噪声): 系统为纯衰减过程，温度异常指数衰减至零，T(t) = T₀·e-λt

• 在无噪声情况下，系统会从初始扰动状态逐渐恢复到平衡状态，这是负反馈的物理作用

• 在现实气候系统中，负反馈代表系统的恢复力（如辐射冷却），噪声代表外部扰动（如天气系统）
